# Course api backend

A REST API project built with Express, Mongoose, Zod, CORS, and dotenv for course management.This project provides a RESTful API built using node js, express and mongoose. Fully crud functionality implemented.

## Features

- Create course
- Get all courss (filtering,sorting & pagination)
- Update course
- Delete course tag
- Update tags
- Data velidation
- User Authentication

## Packges

- express
- mongoose
- zod
- dotenv
- cors
- bcrypt
- jsonwebtoken
